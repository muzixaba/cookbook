#================
# Vim Modes
#================

# Insert Mode
-------------
For editing your file
Press 'i' when in navigation mode
Press 'o' to insert line below and enter insert mode
Press 'O' to insert line above and enter insert mode

# Navigation Mode
-----------------
For navigating the file
Press 'esc' while in insert mode
j,k,h,l - Down, Up, Left, Right

# Undo and Redo
---------------
u - undo
<C-R> - redo

# Exiting Vim
-------------
:w - saves file
:wq - save changes & quit
:q! - quit without saving changes
:q - just quit
:w - saves file without exiting editor
:x - save and exit file

# Navigating a document
----------------------
gg - Go to beggining of doc
G - Go to end of doc
:{n} - go to line
{n}j - go down {n} of lines
{n}k - go up {n} of lines
ctr + d - scroll down
ctr + u - scroll up
{n}gg - jump to line number
{n}j - move down n lines
{n}k - move up n line
zz - scroll selected line to center of screen

# Navigate through words
------------------------
w - move to start of next word
b - move to beginning of word
e - move to end of word
0 - jump to beginning of line
$ - jump to end of line
f{char} - jump to specific character on line (";" or "," next/prev occurance)

# Searching
-----------
In normal mode, type /search-term
Press Enter
n - next occurance
N - previous occurance

# Browse files using Netrw
vim . - opens file tree from inside Vim
i - change list view
:Ex - jump out of file and back into Netrw
:q - close netrw

# Skip block(s) of code
-----------------------
{ - skip backward
} - skip forward

# Delete a line
---------------
dd

# Undo - u
# Redo - ctlr + r

# Add new line
--------------
o - Adds a new line below & puts you inside INSERT mode
O - Adds a new line above current position & puts you in INSERT mode

# Copy & Paste
--------------
yy - copy line
p - paster below

# Do last command
-----------------
.

# Cut and Paste Lines
Be in edit mode
:<start-line-number>, <end-line-number>d
p
